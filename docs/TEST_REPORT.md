## 一、测试概述

### 1.1 测试目的

验证在线聊天系统所有功能模块的正确性、稳定性和性能，确保交付质量。

### 1.2 测试范围

| 模块 | 测试内容 |
|------|---------|
| 用户模块 | 注册、登录、登出、信息修改 |
| 好友模块 | 添加、删除、分组、申请处理 |
| 私聊模块 | 文字、语音、图片、文件、历史 |
| 群聊模块 | 创建、加入、退出、邀请、踢人、消息 |
| 实时通讯 | WebSocket 连接、消息推送、断线重连 |
| 文件传输 | 上传、下载、类型校验、大小限制 |
| 安全 | 密码加密、未登录拦截 |

| 项 | 配置 |

| 操作系统 | Windows 10 |
| JDK | 1.8.0_281 |

#### Bug-01：删除唯一分组导致好友指向不存在分组

**问题描述**：当用户只有 1 个好友分组时，删除该分组，组内好友的 `group_id` 仍指向已删除的 ID。

**严重程度**：中

**复现步骤**：
1. 用户 A 只有 1 个分组"我的好友"
2. 删除该分组
3. 查看 A 的好友列表

**预期**：好友迁移到其他分组（虽然只有 1 个，也应有兜底）

**实际**：好友列表显示异常，提示"分组不存在"

**修复状态**：⏳ 待修复
