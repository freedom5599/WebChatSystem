## 一、系统简介

**在线聊天系统** 是一个仿 QQ / 微信的 Web 版即时聊天工具，支持私聊、群聊、好友管理、文件传输等功能。

**访问地址**：http://localhost:8080

**推荐浏览器**：Chrome 80+、Edge 80+、Firefox 80+

---

## 二、注册与登录

### 2.1 注册新账号

1. 打开 http://localhost:8080
2. 系统自动跳转到登录页
3. 点击「立即注册」
4. 填写：
   - **用户名**（必填，3-50 位，登录账号，唯一）
   - **昵称**（选填，默认与用户名相同）
   - **密码**（必填，6-20 位）
   - **确认密码**（必填）
5. 点击「注册」
6. 注册成功后自动跳转到登录页

### 2.2 登录

1. 打开 http://localhost:8080/login
2. 输入用户名和密码
3. 点击「登录」
4. 登录成功跳转到聊天主界面

## 三、主界面介绍

登录成功后进入聊天主界面，主要分为三块：

```
┌────────────────────────────────────────────────────────┐
│  ┌──────────────┐  ┌────────────────────────────────┐ │
│  │  个人信息     │  │  聊天对象信息  [历史][群成员]    │ │
│  │  [头像][昵称] │  ├────────────────────────────────┤ │
│  ├──────────────┤  │                                │ │
│  │  [好友][群]   │  │                                │ │
│  │  [申请][消息] │  │      消息记录区                │ │
│  │              │  │      （气泡样式）              │ │
│  │  ────好友──── │  │                                │ │
│  │  📁 我的好友  │  │                                │ │
│  │    👤 李四    │  │                                │ │
│  │    👤 王五    │  │                                │ │
│  │  📁 同学      │  │                                │ │
│  │    👤 ...     │  │                                │ │
│  │              │  │                                │ │
│  └──────────────┘  ├────────────────────────────────┤ │
│                    │ [📎][🎤][🖼] [输入框...    ] [发送]│ │
│                    └────────────────────────────────┘ │
└────────────────────────────────────────────────────────┘
```

### 3.1 左侧栏

- **顶部**：显示当前用户头像、昵称、个性签名
  - 点击「编辑」图标可修改个人信息
  - 点击「登出」图标退出登录
- **标签页**：
  - **好友**：显示所有好友（按分组归类）
  - **群聊**：显示所有加入的群
  - **申请**：显示待处理的好友申请和群邀请（小红点提示未读数）
  - **消息**：显示最近的聊天列表

### 3.2 右侧聊天区

- **顶部**：显示当前聊天对象的名称和状态
  - 私聊时显示对方昵称
  - 群聊时显示群名称
  - 在线时显示绿色圆点
- **中部**：消息记录区
  - 自己的消息：右侧，绿色气泡
  - 对方消息：左侧，白色气泡
  - 群消息：左侧，发送者昵称在气泡上方
- **底部**：消息输入区
  - 「📎」上传文件
  - 「🎤」录制语音
  - 「🖼」发送图片
  - 文本输入框
  - 「发送」按钮

## 四、好友管理

### 4.1 添加好友

1. 点击左侧栏顶部「编辑」旁边的「＋」图标
2. 输入对方用户名搜索
3. 在搜索结果中点击「加好友」
4. 填写验证信息（可选）
5. 点击「发送」

**提示**：对方会实时收到通知（无需刷新页面）。

### 4.2 处理好友申请

1. 收到好友申请后，左侧「申请」标签会显示红色数字提示
2. 点击「申请」标签查看详情
3. 点击「同意」或「拒绝」

**提示**：你也会收到对方处理结果的实时通知。

### 4.3 好友分组

- **创建分组**：在「好友」标签页点击「+ 新建分组」按钮
- **移动好友**：右键点击好友 → 选择「移动到」→ 选择目标分组
- **删除分组**：在分组管理中点击「删除」（分组下好友会迁移到默认分组）
- **重命名分组**：右键点击分组名 → 选择「重命名」

### 4.4 删除好友

右键点击好友 → 选择「删除好友」→ 确认。

> 注意：删除好友为双向操作，对方好友列表中也会移除你。

### 4.5 修改好友备注

右键点击好友 → 选择「修改备注」→ 输入新备注名。

## 五、私聊

### 5.1 发起私聊

在左侧「好友」列表中点击任意好友即可开始聊天。

### 5.2 发送消息

在底部输入框中输入文字，按 `Enter` 发送，`Shift+Enter` 换行。

### 5.3 发送文件 / 图片

1. 点击「📎」按钮选择文件
2. 或点击「🖼」按钮选择图片
3. 文件最大 10MB
4. 发送后对方会立即看到

### 5.4 录制语音

1. 点击「🎤」按钮开始录音
2. 再次点击停止
3. 发送语音消息

> 浏览器需授权麦克风权限。

### 5.5 查看历史消息

聊天窗口会自动加载与该好友的历史消息（共 50 条）。

点击聊天对象名称旁的「历史」按钮可加载更多历史消息。

### 5.6 导出聊天记录

点击聊天对象旁的「历史」按钮 → 选择「导出」。

会下载一个文本文件，包含所有聊天记录。

### 5.7 标记已读

打开与某好友的聊天窗口时，该好友发送的所有未读消息会自动标记为已读。

## 六、群聊

### 6.1 创建群组

1. 点击左侧「群聊」标签
2. 点击「+ 创建群」按钮
3. 填写：
   - **群名称**（必填）
   - **群简介**（选填）
4. 点击「创建」

> 创建后你自动成为群主。

### 6.2 通过群 ID 加入群

1. 点击「群聊」标签的「+」按钮
2. 选择「加入群」
3. 输入群 ID
4. 点击「加入」

> 直接加入，不需要群主审批。

### 6.3 邀请好友入群

1. 打开群聊
2. 点击群聊名称旁的「群成员」按钮
3. 点击「邀请好友」按钮
4. 选择好友 → 确认

被邀请人会收到通知，需对方同意后才能加入。

### 6.4 接受 / 拒绝群邀请

1. 收到群邀请后，左侧「申请」标签显示
2. 点击查看详情
3. 点击「接受」加入群，或「拒绝」拒绝

### 6.5 群聊消息

- 进入群聊窗口后即可发送/接收消息
- 群消息会显示发送者昵称
- 群成员都能看到

### 6.6 退出群 / 解散群

- **退出群**：在群聊窗口，点击群名称旁的「群成员」→ 点击「退出群」按钮
- **解散群**（仅群主）：同上，但显示「解散群」按钮（会移除所有成员）

> 注意：群主不能直接退出自己的群，必须先解散或转让（本期未实现转让）。

### 6.7 踢出群成员（仅群主）

1. 在群聊窗口，点击群名称旁的「群成员」
2. 找到要踢出的成员
3. 点击「踢出」按钮

## 七、消息推送机制

### 7.1 实时通知场景

| 事件 | 通知方式 |
|------|---------|
| 收到好友申请 | WebSocket 实时推送 + 弹窗提示 |
| 好友申请被处理 | WebSocket 实时推送 + 弹窗提示 |
| 收到群邀请 | WebSocket 实时推送 + 弹窗提示 |
| 私聊消息 | WebSocket 实时推送 |
| 群聊消息 | WebSocket 实时推送 |
| 好友上线/下线 | WebSocket 实时推送（侧边栏状态点变化） |

### 7.2 离线消息

- 如果对方不在线，私聊消息会存入数据库
- 对方下次登录后可通过历史消息查看

### 7.3 网络断线重连

WebSocket 断线后会自动重连（5 秒后），无需手动操作。

## 十、常见问题

### Q1：消息发不出去？
- 检查网络连接
- 确认对方是否已登录
- 刷新页面重试

### Q2：上传文件失败？
- 确认文件 < 10MB
- 确认文件类型（不支持的文件类型会被拒绝）

### Q3：麦克风无法使用？
- 检查浏览器是否授权麦克风权限
- 在浏览器设置中允许该网站使用麦克风

### Q4：对方看不到我的消息？
- 确认对方也登录了
- 刷新页面（重新建立 WebSocket 连接）
- 查看浏览器控制台是否有错误

### Q5：怎么注销账号？
- 本系统未提供注销功能。如需删除账号，请联系管理员。
